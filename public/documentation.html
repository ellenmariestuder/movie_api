myFlix App Documentation

myFlix is a web application for movie enthusiasts, providing users with access to information about movies, directors, and genres. Users can also create a myFlix account, where they can store a list of their favorite movies.


Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET None A JSON object holding data about all the movies
Return data about a single movie title /movies/[Title] GET None A JSON object holding data about a single movie title, containing description, genre, director, image URL, and whether it's featured or not. Example:
{
Title: Two Weeks Notice
Description: 2002 American romantic comedy film starring Hugh Grant and Sandra Bullock.
Genre: Romantic Comedy
Director: Marc Lawrence
ImageURL: twoweeksnoticemovie.jpeg
Featured: Yes
}
Return data about a genre by name/ title genres/[Name] GET None A JSON object containing a description of the genre
Return data about a director by name directors/[Name] GET None A JSON object holding data about a single director, containing bio, birth year, and death year. Example:
{
Name: Marc Lawrence
Bio: Marc Lawrence is an American director, screenwriter, and producer. Lawrence is best known for his numerous collaborations with Sandra Bullock and Hugh Grant.
Birth year: 1959
Death year:
}

Allow users to remove a movie from their list of favorites /users/[Username]/Movies/[MovieID] DELETE None A text message confirming the movie has been removed
